# :radiant-text[About Radiant]

**Radiant (璀光)** 是一个专为 Nuxt 4 设计的静态博客主题。它旨在提供一种“冷淡”、“纯净”的阅读体验。

设计灵感来自于玻璃拟态（Glassmorphism）和极简主义。我使用了低饱和度的色彩、半透明的背景模糊效果以及精致的排版，来构建一个既现代又舒适的数字空间。

### 主要特点

- ✨ **Nuxt Content v3** 驱动
- 🎨 **Tailwind-like** 自定义 CSS 变量
- 📱 **完全响应式** 设计
